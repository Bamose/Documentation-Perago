## AcceptInvitation

Interface for invitation acceptance data.

**Fields**

- `personnelId` - ID of invited user
- `password?` - Optional password if new account
- `accountExists` - Flag if existing account

## AccessTokenRequest

Interface for access token request data.

**Fields**

- `username` - User's username
- `password` - User's password
- `scope?` - Optional scope parameter
- `client_id?` - Optional client ID
- `client_secret?` - Optional client secret
- `grant_type?`- Optional grant type

## LoginCredential

Interface for login credential data.

**Fields**

- `username` - User's username
- `password1`- User's password

# session

Interface for session data.

**Fields**

- `accessToken` - Access token
- `accessTokenExpriresInMinutes` - Access token expiration time
- `accessTokenIssueDate` - Access token issue date
- `organizations` - User's organizations
- `profile` - User profile
- `refreshToken` - Refresh token
- `refreshTokenExpriresInDays` - Refresh token expiration time
- `refreshTokenIssueDate` - Refresh token issue date
- `selectedOrganization?` - Selected organization
- `selectedUnit?` - Selected unit

## SessionStatus

Enum for session authentication status.

**Values**

- `IS_CHECKING` - Initial status while validating session
- `IS_LOGGED_IN` - Session is authenticated
- `IS_LOGGED_OUT` - Session is not authenticated
- `IS_PROMPTING_LOG_OUT` - Logging out existing session
